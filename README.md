# ☕ Java Fundamentos & Arquitetura

Repositório dedicado ao reforço dos meus fundamentos de desenvolvimento de software através da linguagem **Java**.

---

## 🎯 Objetivo
Meu foco principal é utilizar a robustez do Java para consolidar conhecimentos em **arquitetura de sistemas, desempenho e boas práticas**. Utilizo meu domínio em **TypeScript e NestJS** como base para comparar paradigmas e explorar como o Java torna conceitos de backend rigorosos e explícitos.

Este projeto registra a aplicação de:
* **Fundamentos de Engenharia:** Lógica estruturada e algoritmos.
* **Pensamento Arquitetural:** Organização de código e estruturas de dados eficientes.
* **Padrões de Projeto:** Aplicação nativa de princípios de Orientação a Objetos (OO) para soluções escaláveis.

## 📁 Estrutura do Repositório
O código é organizado por pacotes que representam desafios técnicos e cenários de backend, com foco total em lógica de negócio e manutenibilidade.

## 💡 Evolução e Paradigmas
O contato com o ecossistema **Java** em paralelo ao **Node.js/TypeScript** me permite uma visão crítica sobre qual tecnologia melhor resolve cada problema de engenharia. O objetivo aqui é dominar a construção de aplicações robustas, aproveitando a tipagem forte e a maturidade da JVM para elevar o nível das minhas entregas no backend.
